# python-ftpClient
A ftp client for Linux

## **Dependencies**

python 3.x

tqdm 
run: <br>
```
pip install tqdm
```

## **Usage**

```
python3 ftp-client.py ip port
```

If you want to connect as anonymous user leave the username and password blank
#### **Notes**
works on termux on android and with the ubuntu terminal on windows
